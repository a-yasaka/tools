□ このソフトは何者か
  Proxy On/Off er3 (以下 "PONF3")は、Internet Explorerのプロキシ設定を手軽にON/OFFするためのソフトです。
  プロキシの設定値を変更するソフトは多数ありますが、やりたいことはON/OFFだけだったので(探してもなかなか見つからないので)作りました。
  気軽に「ぽんふさん」と呼んでください。

  通常はInternet Explorerのメニューから 
  「ツール(T)→インターネットオプション(O)→接続→LANの設定(L)→プロキシサーバー」のチェックボックスをクリックしてから
  「OK→OK」としないといけないところを、最短ツークリックで切替出来ます。まあ便利。

  ONになると緑インジケーターが点灯して、OFFになると赤が点灯します。
  両方点灯しているときor両方消灯しているときは、うまいこと状態を取得できなかった時です。
  (表示は必ずしもその時点での実際のIE設定と同じとは限りません。当たり前ですが、表示後に設定を変えるとズレます。)

  挙動としては、当該レジストリを変更してIE設定をリロードしているだけです。


□ 動作環境
  .Net Framework 2.0が動くWindows XP以降のWindows OS。
  その他、動く組み合わせと動かない組み合わせは多々あると思います。
  Windows8以降の場合は、先に ".Net Framework 3.5 (.NET 2.0および3.0を含む)" を有効化して下さい。
  (例：DISM /Online /Enable-Feature /FeatureName:NetFx3 /All /LimitAccess /Source:X:\sources\sxs)

  本バージョンは主に下記の環境で動作確認をしています。
  ・Windows 7 Home Basic /w Service Pack 1 (x64)
  ・Windows 7 Professional /w Service Pack 1 (x86)

  本ソフトは下記の環境で動作報告があります。 (大きな変更のない旧バージョンを含む)
  ・Windows XP Home Edition /w Service Pack 3 (x86) ※1
  ・Windows 7 Professional /w Service Pack 1 (x64)
  ・Windows 8.1 /w Update (x64) ※2
  ・Windows Server 2012 R2 Standard /w Update (x64) ※2

  ※1:.Net Framework 2.0の含まれるランタイムをインストールしています。
  ※2:".Net Framework 3.5 (.NET 2.0および3.0を含む)" を有効化しています。


□ 使い方
  ◇ 起動
    起動するとタスクトレイに潜り込みます。

  ◇ 切替
   アイコンをダブルクリックすると、ON/OFFを順番に切り替えます。
   右クリックするとメニューが出て来ます。
   バージョン情報以下のメニューは、選択出来ますが何も設定していません。表示用です。

  ◇ 表示
   「バルーンを表示する」にチェックを入れると、変更毎に通知が出ます。(デフォルトON)
   ウザい場合はOFFに出来ます。


□ メニュー構成
  メニュー
  ┣ バージョン情報             … 子項目にバージョン情報を表示します。
  ┃ ┗ 各情報                  … 表示用項目なので、選択しても何もありません。
  ┣ 開く                       … コントロールパネルの項目を開くためのメニューです。子項目を選んでください。
  ┃ ┗ 各子項目                … 項目を開きます。環境によっては開けないことがあるかもしれません。
  ┃                               ----------------
  ┣ バルーンを表示する         … 変更毎にバルーン表示をします。
  ┃                               ----------------
  ┣ PROXY ON                   … プロキシをONにします。
  ┣ PROXY OFF                  … プロキシをOFFにします。
  ┃                               ----------------
  ┗ 終了                       … PONF3を終了します。


□ その他挙動など
  ・起動時にini読み込み、終了時にini保存をします。
  ・iniファイルが無い場合は、終了時にiniファイルを作成します。


□ アンインストール
  そのままディレクトリごと削除して下さい。


□ 仕様・制限事項
  ・全てのWindows OSとIEの組み合わせで動作確認が取れているわけではありません。やりたいですが割と無理です。
    なので、動かない環境があったらそれが非動作環境です。
  ・マウスカーソルを合わせてもタイトルなどは表示されません。バルーンと重なって見づらかったのでやめました。
